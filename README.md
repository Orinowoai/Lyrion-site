# LYRĪON

**When the Stars Learn Your Silhouette**

LYRĪON is a celestial e-commerce platform offering mystic apparel, accessories, and homeware as spiritual instruments of self-realization and aesthetic alignment.

## Features

- **90+ Celestial Products**: Zodiac-themed apparel, accessories, and homeware
- **Oracle Glimpse Generator**: Personalized celestial insights with seeded randomness
- **Dynamic Catalog**: Sign-based filtering and seasonal collections
- **Ritual Bundles**: Curated celestial sets aligned with astrological events
- **Netlify Forms**: Integrated contact and oracle forms
- **Blog Content**: Astrology and design editorial content

## Structure

- `/data/` - JSON data files (catalog, astrology metadata)
- `/js/` - JavaScript functionality (catalog, oracle-glimpse)
- `/css/` - Styling
- `/assets/` - Images and media
- `/posts/` - Blog content (markdown)
- `*.html` - Site pages (index, shop, collections, oracle, contact, about)
